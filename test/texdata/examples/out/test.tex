\documentclass{article} \usepackage{amsmath} \usepackage{amssymb} \usepackage[left = 20mm,right = 20mm]{geometry} \begin{document}\begin{huge}The Dirac delta function $ \delta (x) $ satisfies: \begin{enumerate}\item{\[ \delta (x) = 0 \, \text{for all $ x \neq 0 $} \]} \item{\[ \int_{- \infty}^{\infty}\delta (x)dx = 1 \]} \end{enumerate} From these properties, it follows that for all function $ f, $ \[ \int_{- \infty}^{\infty}\delta (x) f(x) dx = f(0). \] \end{huge} \end{document}
