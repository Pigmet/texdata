\documentclass{article} \usepackage{amsmath} \usepackage{amssymb} \usepackage[left = 20mm,right = 20mm]{geometry} \begin{document} The Dirac delta function $ \delta (x) $ satisfies \[ \left\{ \begin{array}{c} \delta (x) = 0 \; \text{if $ x \neq 0 $} , \\ \int_{- \infty}^{\infty}\delta (x)dx = 1 . \end{array} \right. \] And for any function $ f(x), $ we have the equality \[ \int_{- \infty}^{\infty}\delta (x) f(x)dx = f(0) . \] \end{document}
